Instructions:
> Build and run the jar file
> Tools > Settings and fill in the required data and press Update
Note: Item Codes field should contain values in CSV format as follows:
ITEM_1,ITEM_2,ITEM_3,...

Save the config changes by pressing Update
When ready, press Start from the main frame, the GUI should begin
to populate, the console output shows the progress,
Every couple of minutes, the program will ping the AmiAmi website
looking for the specified items
